This repository contains the source code of the UniHome Website, developed as part of a Diploma Thesis at the University of the Aegean, Department of Information and Communication Systems Engineering.
The project was created by the students: Evangelos Avtzis, Athanasios-Ioannis Gkonos

UniHome is a web-based platform designed to support smart housing management and related services through a modern web interface and a scalable backend architecture.
The system is built using a full-stack approach and containerized for easy deployment and portability.

Technologies Used

Frontend: Angular
Backend: Spring Boot
Database: MySQL
Containerization: Docker

Deployment

The application is designed to run using Docker containers, allowing all services (frontend, backend, and database) to be deployed and managed efficiently through Docker Compose.

This project was developed for academic purposes as part of a diploma thesis and demonstrates the integration of modern web technologies, microservice architecture principles, and containerized deployment.
